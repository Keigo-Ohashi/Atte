# Atte

勤怠管理システム

## 作成した目的

勤怠管理を自動化するため

## 機能一覧

・アカウント登録(メール認証)
・出退勤と休憩打刻
・日付別勤怠表
・アカウント一覧
・アカウントごとの勤怠表

## 仕様技術

Laravel 8.x
